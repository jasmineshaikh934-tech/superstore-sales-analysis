# Superstore Sales Data Analysis

This project analyzes a retail sales dataset to understand business performance across different product categories, regions, and customer segments.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Technology category generates the highest revenue
- West region has the highest sales
- High discounts reduce profitability

## Project Files
- analysis.ipynb – Data analysis notebook
- SuperStoreOrders.csv – Dataset
